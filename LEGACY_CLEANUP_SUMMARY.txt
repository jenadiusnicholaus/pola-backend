## Current API Structure

All hub features now use the **unified API**:

### Unified Endpoints (Active):
- `/api/v1/hubs/content/` - All hub content (filter by `?hub_type=advocates|students|forum`)
- `/api/v1/hubs/comments/` - All comments
- `/api/v1/hubs/lecturer-follows/` - Follow system
- `/api/v1/hubs/questions/` - Q&A system
- `/api/v1/hubs/messages/` - Private messaging

### Legacy Endpoints (Removed):
- ❌ `/api/v1/hubs/advocates/posts/`
- ❌ `/api/v1/hubs/students/materials/`
- ❌ `/api/v1/hubs/forum/discussions/`
